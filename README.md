# 数据驱动模型
Vue 的执行流程  
   1. 获得模板: 模板中 '有坑'
   2. 利用Vue搞糟函数中所提供的数据来填坑，得到可以在页面中显示的 '标签'了
   3. 将标签 特换页面中原来有坑的标签
   Vue 利用我们提供的数据和 页面中 模板，生成了一个新的html标签(node 节点)，替换到了页面中防止模板的位置
